배열 이해하기

배열이란, 목록의 나열이란 의미를 가지고 있다.
이는 다양한 내용, 변수, 함수를 하나에 나열할 수 있는 방식이다.

배열의 예시 : 카페의 메뉴

  • [0] 아메리카노
  • [1] 카페라떼
  • [2] 카페모카
  • [3] 홍차
  • [4] 밀크티

객체 이해하기

객체란, 배열과 유사하며, 나열한다는 것에서 의미는 비슷하지만,
정의와 설명이 첨부된 형식이다.
객체의 정의를 함수로 처리 되면 그를 메소드라고 불리우며, 단순 설명으로 처리될 시엔 프로퍼티라고 한다.

객체의 예시 : 음료의 종류

커피 :
원두를 주 원료로 한 음료
쥬스 :
각종 과일을 재료로 하는 음료
홍차 :
녹찻잎을 삭여 우려내서 만드는 음료
