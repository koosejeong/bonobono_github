js 기초 이해하기

javascript란?

자바스크립트란, 객체 기반의 스크립트 프로그래밍 언어
주로 웹에 특화 되어 있는 프로그래밍 언어다. 과거 자바의 아류이자 초급 언어라는 불명예 적인 말도 들었지만,
현재(2019년)는 가장 인기 있는 언어 중 하나이다.

객체 기반이며, 객체뿐 아니라 함수도 지향하는 일급 객체로 불리우는 언어이다.
과거 웹에서 html, css만 제어하던 때를 넘어 현재는 앱 개발도 가능하며,
서버를 컨트롤 하기도 하는, 다양한 분야에서 사용 가능한 고급 언어이다.

자바스크립트는 ECMAscript 의 표준 언어 체계를 따르고 있으며, ECMA 표준은 매년 6월 17일에 새로운 표준을 선언하고 있다.

유사 언어로는 Type script, Coffee script, Dart, Elm , Action script 등이 있다.
Coffee script와 Action script는 요새 사용하지 않는다.

자바스크립트에서 인기 있는 라이브러리, 프레임 워크로는
jQuery, Angularjs, Reactjs, VueJs, ember, polymer, handlerbarJs, d3Js....등이 있다.
